## Overview

The IPE Framework (Intake-Process-Expression) provides temporal dynamics for the LJPW coordinate system when applied to living systems. While LJPW describes *where* a system is in semantic space, IPE describes *how* living systems move through that space.

| Framework | Describes | Type |
|-----------|-----------|------|
| **LJPW** | System state (coordinates) | Spatial |
| **IPE** | System flow (trajectory) | Temporal |
| **Integration** | State + trajectory | Complete dynamics |

## The IPE Framework

### Definition

| Phase | Description | Direction |
|-------|-------------|-----------|
| **Intake** | Receive energy and information from environment | Outside → Inside |
| **Process** | Transform and integrate within system | Within boundary |
| **Expression** | Manifest in behavior, growth, output | Inside → Outside |

IPE describes **boundary dynamics** — how living systems exchange with their environment across self-defined boundaries.


### The 2+2 Structure in IPE

The LJPW 2+2 structure (P-W fundamental, L-J emergent) maps directly:

- **Process phase = P-W dynamics** — The fundamental pair operating within the system
- **Intake and Expression = L-mediated** — Emergent connectivity enabling boundary crossing
- **J governs all phases** — Balance regulation, not a phase itself

### J as Cycle Governor

Justice does not appear as its own phase because **J regulates all phases**:

```
INTAKE:     J prevents over-accumulation
            ├── Feedback inhibition
            ├── Receptor desensitization
            └── "Don't take more than you can process"

PROCESS:    J maintains stoichiometric balance
            ├── Enzyme kinetics equilibria
            ├── Metabolic regulation
            └── "Transform proportionally"

EXPRESSION: J prevents over-expression
            ├── Output limits
            ├── Secretion regulation
            └── "Don't give more than is needed"
```
